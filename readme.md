# Qute

A simple website hosts surveys for student activities, built with Django.   
基于Django实现的简明答题网站，主要用于学生活动答题。   

## 功能

- 输入合理的学号与姓名参与答题
- 支持各种选择题与文本题目
- 前端自动保存答题进度
- 选择题自动判卷
- 后端支持热切换两套不同题目，前端彻底隔离
- 管理员后台查看答题结果

## 安装

0. 安装 Python 3.8 及以上版本（或者更低，只要你让django能跑起来）
1. 克隆本仓库
2. 创建并激活虚拟环境并安装依赖`pip_requirements.txt`
3. 运行数据库迁移（详见Django文档）
4. 运行`server`启动服务器

## 实现

- 前端使用纯 HTML/CSS/JavaScript 编写（ai真是太好用了.jpg）
- 后端使用 Django 框架编写（很好用）
- 数据库存储使用默认的 SQLite（最多几千个条目闹麻了）
- 基本处理了所有边界情况与异常情况，应该能防范除DDOS以外的大部分攻击（我们学校神人很多）
- 没有做过分的抽象与封装，代码结构比较平铺直叙，在够用的基础上保证了一定的简洁（假的，只是代码量不大所以没那么史山）
